# AI Interview Coach

## Overview
AI-powered interview preparation platform using Resume Analysis, Mock Interviews and AI Evaluation.

## Features
- Resume Parsing
- Skill Extraction
- AI Resume Feedback
- Technical Questions
- HR Questions
- Mock Interview
- Voice and Text Input
- AI Evaluation
- PDF Report Generation

## Tech Stack
- Python
- Flask
- Groq API
- Gemini API
- HTML
- CSS
- JavaScript

## Run Project
```bash
pip install -r requirements.txt
python app.py
```
